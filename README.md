# Signal Oracle - 운명 정밀 종합 검진

React + Vite + Tailwind CSS로 구현된 심리 테스트 애플리케이션입니다.

## 기능

- 6개 영역(직관, 금전, 직업, 신체, 관계, 애정) 30문항 테스트
- 실시간 진행률 표시
- 결과 분석 및 레이더 차트
- 결과 공유 기능

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 기술 스택

- React 18
- Vite
- Tailwind CSS
- Lucide React (아이콘)
